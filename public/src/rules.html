How Is The Token Issued ?

Tokens are earned through Flipkart purchases, via referrals and social media engagement. Eco-friendly products will be rewarded with an augmented token earning.

How Is The Token Regulated ?

Number of tokens will be limited for a user with a limited inflation rate that would incentivize users to constantly trade tokens as opposed to hoarding them. The Tokens would start decaying if a user crosses a certain amount tokens followed by an inactivity .This creates a dynamic ecosystem that promotes circulation of currency.

How Can Tokens be Spent?

Token-Conversion: Users can convert FlipKash to loyalty points directly. Which can be redeemed on various subsidiary platforms. Perks: Users can get discounts, free shipping, first access to new products and combo offers by the use of tokens for purchases. Charity: Users can donate to charities using cryptocurrency donations.
